## How to Start
1. **git clone server** 
    1. `git clone https://github.com/JENOSS/com-partz.git`
2. **set IntelliJ**
    1. select File → Project Structure → Project
        1. select Java SDK 17
    2. select Gradle tap
        1. select Tasks → other → compileJava
        2. select Tasks → application → bootRun
3. **pod install**
